# WidgetSteps1

<SourceLink name="WidgetSteps1" />

The **Steps 1** component is a versatile widget for presenting sequential or categorized information in a visually engaging way. It is ideal for guiding users through processes, showcasing organizational highlights, or displaying key actions.

## Props

<div className="overflow-x-auto">
  <div className="min-w-xl">
    | Prop | Type | Default | Description |
    |------|------|---------|---------|
    | `title` | `string` | - | `Title` in [Headline](/docs/headline) component |
    | `subtitle` | `string` | - | `Subtitle` in [Headline](/docs/headline) component |
    | `tagline` | `string` | - | `Tagline` in [Headline](/docs/headline) component |
    | `asHeader` | `h1` \| `h2` \| `h3` \| `h4` \| `h5` \| `h6` | `h2` | HTML heading tag to use for the title |
    | `asSubtitle` | `h1` \| `h2` \| `h3` \| `h4` \| `h5` \| `h6` \| `p` \| `span` \| `div` | `p` | HTML tag to use for the subtitle |
    | `items` | `Array` | `[]` | Array of [Item](/docs/types#item) items |
    | `callToAction` | `ToAction \| string \| ToAction[] \| string[]` | - | Action button or link. See the [ToAction](/docs/types#action-buttons) for details.  |
    | `isReversed` | `boolean` | `false` | Defines left or right position |
    | `isAfterContent` | `boolean` | `false` | - |
    | `id` | `string` | - | Specify a unique id for the section |
    | `isDark` | `boolean` | `false` | Adds the `dark` CSS class to the [WidgetWrapper](/docs/feature-widget-wrapper) |
    | `bg` | `string` | - | Set background element |
    | `classes.container` | `string` | - | CSS classes for the root container |
    | `classes.headline.{...}` | `string` | - | CSS classes for the [Headline](/docs/headline) component |
    | `classes.itemIcon` | `string` | `bg-card text-card-foreground` | - |
    | `classes.itemTitle` | `string` | `` | - |
    | `classes.itemDescription` | `string` | `` | - |
  </div>
</div>
