## Props

<div className="overflow-x-auto">
  <div className="min-w-xl">
    | Prop | Type | Default | Description |
    |------|------|---------|---------|
    | `position` | `center` \| `left` \| `right` | `center` | - |
    | `links` | [ILink](/docs/types#ILink)[] | `[]` | - |
    | `actions` | [ICallToAction](/docs/types#ICallToAction)[] | `[]` | - |
    | `isSticky` | `boolean` | `false` | - |
    | `isDark` | `boolean` | `false` | Adds the `dark` CSS class to the wrapper |
    | `showToggleTheme` | `boolean` | `false` | - |
    | `scrollOffset` | `number` | `60` | How much do you need to scroll the page to apply `pinned` classes |
    | `classes.container` | `string` | `max-w-7xl md:mx-auto` | CSS classes for the wrapper container |
    | `classes.links` | `string` | `hover:text-link dark:hover:text-white` | - |
    | `classes.dropdownLinks` | `string` | `md:hover:bg-gray-200 hover:text-link dark:hover:text-white dark:hover:bg-zinc-800` | - |
    | `classes.darkModeIcon` | `string` | - | - |
    | `classes.pinned` | `string` | `bg-background md:bg-background/90 md:backdrop-blur-md` | Classes that will be applied when scrolling the page |
  </div>
</div>
