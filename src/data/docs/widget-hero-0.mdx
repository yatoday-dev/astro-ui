## Props

<div className="overflow-x-auto">
  <div className="min-w-xl">
    | Prop | Type | Default | Description | 
    |-----------|------------|------------|------------| 
    | `title` | `string` | - | `Title` in [Headline](/docs/feature-card-1) component | 
    | `subtitle` | `string` | - | `Subtitle` in [Headline](/docs/feature-card-1) component | 
    | `tagline` | `string` | - | `Tagline` in [Headline](/docs/feature-card-1) component | 
    | `columns` | `number` | `1` | Number of columns in the grid. From 1 to 4 | 
    | `id` | `string` | - | Specify a unique id for the section | 
    | `isDark` | `boolean` | `false` | Adds the `dark` CSS class to the wrapper |
    | `bg` | `string` | - | Set background element |
    | `classes.container` | `string` | - | CSS classes for the root container | 
    | `classes.headline.container` | `string` | - | CSS classes for the `container` in the [Headline](/docs/feature-card-1) component |
    | `classes.headline.title` | `string` | - | CSS classes for the `title` in the [Headline](/docs/feature-card-1) component | 
    | `classes.headline.subtitle` | `string` | - | CSS classes for the `subtitle` in the [Headline](/docs/feature-card-1) component | 
    | `classes.headline.tagline` | `string` | - | CSS classes for the `tagline` in the [Headline](/docs/feature-card-1) component |
  </div>
</div>
