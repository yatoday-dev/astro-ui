## Props

<div className="overflow-x-auto">
  <div className="min-w-xl">
    | Prop | Type | Default | Description |-----------|------------|------------|------------| | `id` | `string` | - |
    Set to specify a unique id for an HTML element | | `as` | `string` | `section` | HTML tag name for the wrapper | |
    `containerClass` | `string` | - | Additional CSS classes to apply to the wrapper | | `isDark` | `boolean` | `false`
    | Adds the `dark` CSS class to the wrapper | | `bg` | `string` | - | Set background element |
  </div>
</div>
