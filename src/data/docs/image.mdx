## Usage

```astro
---
import { Image } from 'yt-astro-ui';
import myImage from '~/assets/images/local_placeholder.jpeg';
---

<!-- Default -->
<Image src="~/assets/images/local_placeholder.jpeg" alt="Local Image" />

<!-- Imported image -->
<Image src={myImage} alt="Local Image" />

<!-- With props -->
<Image
  src={src}
  class="w-full md:h-full bg-gray-400 dark:bg-slate-700"
  widths={[400, 900]}
  width={400}
  sizes="(max-width: 900px) 400px, 900px"
  alt={title}
  aspectRatio="16:9"
  layout="cover"
  loading="lazy"
  decoding="async"
/>
```

### layout

Helper attribute for setting image size

- **constrained** - If screen is wider than the max size, image width is the max size, otherwise it's the width of the screen <br />Will be rendered with: `(min-width: ${width}px) ${width}px, 100vw`
- **fixed** - Image is always the same width, whatever the size of the screen. Will be rendered with: `${width}px`
- **fullWidth** - Image is always the width of the screen. Will be rendered with: `100vw`
